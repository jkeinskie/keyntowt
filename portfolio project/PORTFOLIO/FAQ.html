How do you optimize the performance of a website?

- Performance optimization involves techniques like minification of CSS/JS,
image compression, lazy loading, and utilizing a Content Delivery Network
(CDN).

What are the new features in HTML5 and CSS3?

- HTML5 introduces new semantic elements, local storage, and native multimedia
support. CSS3 includes features like transitions, animations, and flexbox/grid layouts.

What is the DOM (Document Object Model)?

- The DOM is a programming interface that represents the structure of a document
as a tree of objects, enabling dynamic access and manipulation of document content.
